# 旋转地球仪

一个使用 Three.js 实现的交互式 3D 旋转地球仪。

## 功能特性

- 3D 地球渲染，带有陆地和海洋纹理
- 动态云层效果
- 星空背景
- 自动旋转动画
- 鼠标拖拽旋转
- 鼠标滚轮缩放
- 移动端触摸支持
- 响应式设计

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000

   # 然后访问 http://localhost:8000
   ```

## 控制说明

- **拖动鼠标**：旋转地球
- **滚轮**：缩放视图
- **暂停/继续**：控制自动旋转
- **显示/隐藏云层**：切换云层显示
- **重置视图**：恢复初始视角

## 技术栈

- Three.js - 3D 图形库
- JavaScript (原生)
- HTML5 / CSS3

## 浏览器支持

支持所有现代浏览器，包括：

- Edge
